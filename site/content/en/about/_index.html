--- title: About Amazon Genomics CLI linkTitle: About menu: main: weight: 40 --- {{< blocks/cover title="About Amazon Genomics CLI" image_anchor="bottom" height="min" >}} {{< /blocks/cover >}} {{% blocks/section %}} Amazon Genomics CLI is an open-source command line tool developed by the AWS Health AI team to simplify running genomics workflows in the cloud. Amazon Genomics CLI takes care of deploying and managing the cloud infrastructure required to run workflows in common bioinformatics workflow languages such as WDL and Nextflow DSL allowing you to focus on analytics and workflow development. {{% /blocks/section %}}
